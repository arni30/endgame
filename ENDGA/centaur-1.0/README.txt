Name:
  Centaur
Version:
  1.0
Description:
  Centaur sprites created for Stendhal. Based on "[LPC] Horses" by
  bluecarrot16 & "24x32 bases" by Svetlana Kushnariova (Cabbit).
Sprite details:
  - Orientation: orthogonal
    - N/E/S/W
    - I will eventually add S/W/E/N versions.
  - Dimensions: 64x85
  - Animation:
    - idle (center frames)
    - walking: 12 frames (3 per direction)
    - Animated preview @ 150ms delay.
  - PNG images optimized with indexed color (smaller files).
  - GIMP sources (.xcf) use RGB color.
